第一章：Quantumult是什么？

Quantumult是一款专为iOS平台设计的网络代理工具，其主要功能包括网络代理、规则管理、流量监控和隐私保护。与其他同类工具相比，Quantumult以其轻量级、高效率和强大的自定义能力脱颖而出。

1.1 Quantumult的核心功能

  1. 多协议代理支持
    Quantumult支持HTTP、HTTPS、SOCKS5等多种代理协议，用户可以根据需求选择最适合的代理方式。无论是科学上网还是网络加速，Quantumult都能提供灵活的解决方案。

  2. 智能规则管理
    用户可以自定义网络请求规则，例如指定某些网站走代理，而其他网站直连。这种精细化的控制能力让Quantumult在复杂网络环境中游刃有余。

  3. 实时流量监控
    Quantumult提供详细的流量统计功能，用户可以实时查看网络使用情况，避免流量超标或异常消耗。

  4. 隐私保护
    通过代理服务器隐藏真实IP地址，Quantumult有效防止了网络追踪和数据泄露，为用户提供了更高的隐私安全性。

FAQ

V2Ray 的传输协议选择对性能有何影响？
V2Ray 支持 TCP、WebSocket、gRPC、QUIC 等多种传输协议。不同协议在延迟、抗封锁和吞吐量上差异明显。用户可根据网络环境选择最优协议，以获得稳定且高速的连接体验。
VLESS + Reality 与 Trojan 在性能上差异大吗？
两者性能相近。VLESS + Reality 伪装更自然，适合强封锁地区；Trojan 则依托标准 TLS 实现高速连接。选择取决于服务器支持与环境稳定性。
QuantumultX 如何实现按应用分流？
QuantumultX 可以根据不同应用程序的流量类型设置分流规则，通过策略组选择最优节点。这样可保证游戏、视频和浏览器流量经过最优节点，提高访问速度和网络稳定性。
QuantumultX 的广告屏蔽原理是什么？
QuantumultX 使用分流规则与脚本结合的方式实现广告屏蔽。通过重写 HTTP 请求和阻止特定域名，它能有效去除 App 内广告或网页弹窗，提升浏览体验。
Trojan 使用自签名证书会带来哪些问题？
自签名证书无法被客户端默认信任，可能导致连接失败或需要手动安装证书。它也容易被中间人伪造识别，不利于长期生产环境安全部署，建议使用可信 CA 证书以获得更好兼容性与安全性。
Hiddify Next 的多用户限速功能如何使用？
Hiddify Next 支持在管理面板中为每个用户设置带宽上限。管理员可以按需分配速率或流量配额，防止单一用户占用过多资源，从而保持整体网络稳定。
如何用 iptables 限制 V2Ray 服务的访问来源？
使用 iptables 规则仅允许可信 IP 或 CDN 的中转 IP 访问服务端口，例如 `iptables -A INPUT -p tcp -s --dport 443 -j ACCEPT`，并在默认策略下拒绝其他来源，从而降低被扫描与滥用风险。
Potatso Lite 是否可以手动管理多个节点？
支持。用户可以手动输入节点参数或扫描二维码导入节点，并可进行修改或删除。虽然不支持订阅自动更新，但足够满足轻量用户对节点管理和分流的基本需求。
V2Ray 的 outbound freedom 模式有什么作用？
freedom 是直连模块，可用于分流策略中实现部分流量直连。用户可在 routing 中设置匹配规则，将国内域名或局域网 IP 指向 freedom 出口，从而减少代理开销和延迟。
Matsuri 节点排序如何提高视频播放流畅性？
Matsuri 会测速节点延迟，并结合策略组自动选择延迟低、稳定性高的节点处理视频流量。减少缓冲和卡顿，保证在线视频播放顺畅，提高整体网络体验。
